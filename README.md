# PhotoCompress (dulu: kompres-foto-jkt48)

## Cara pakai (lokal)
1. Clone repo
2. Jalankan `npx serve` atau buka `index.html` via static server
3. Seret foto ke area upload → atur quality & max width → klik Kompres → Download ZIP

## Deploy
- Push repo ke GitHub dan aktifkan GitHub Pages (branch `main`/`gh-pages`).
